
# Rock Paper Scissors Game (C++)

A robust implementation of the classic "Rock Paper Scissors" game developed as part of my programming journey with the 
**Dr.Mohammed Abu-Hadhoud Roadmap (Level 5)**.
 This project focuses on logic building, game flow management, and clean code structures.

## 🚀 Features
- **Interactive Gameplay:** Play against a smart computer opponent.
- **Round Selection:** Choose how many rounds you want to play in a single game session.
- **Score Tracking:** Real-time tracking of Player vs. Computer wins and draws.
- **Game Summary:** A detailed final result screen showing the overall winner.
- **Colorful Console:** Uses visual feedback to enhance the player experience.

## 🧠 What I Learned
During the development of this project, I practiced:
- Advanced **Enums** and **Structs** to manage game states.
- Complex **Conditional Logic** and nested loops.
- Input validation to ensure a smooth user experience.
- Code modularity by breaking down the game into reusable **Functions**.

## 🛠️ How to Run
1. Clone the repository or download the `Source.cpp` file.
2. Open it in any C++ IDE (like Visual Studio, Code::Blocks, or Cxxdroid).
3. Compile and Run the program.
4. Follow the on-screen instructions to play!
